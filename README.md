## `$ sobre o projeto`
 
```bash
$ Para Auxiliar Novos Úsuarios e Futuros Programadores 
```
 
Site educacional que ensina, na prática, como construir aplicações web
completas **sem nenhum framework** — só `HTML`, `CSS` e `JavaScript` puro.
 
O objetivo é mostrar que dá pra fazer roteamento, manipulação de DOM,
consumo de API, animações e até persistência de dados (`localStorage`)
usando apenas o que o navegador já oferece.
 
---
 
## `$ o que você vai aprender`
 
```bash
$ ls modulos/
```
 
- ✅ Manipulação de DOM sem `querySelector` de "mão pesada"
- ✅ SPA (Single Page Application) com roteamento via JS puro
- ✅ Componentização usando funções e template strings
- ✅ Consumo de API com `fetch`
- ✅ Persistência de dados com `localStorage`
- ✅ Animações com `requestAnimationFrame` e CSS
- ✅ Boas práticas de organização de código (clean code)
